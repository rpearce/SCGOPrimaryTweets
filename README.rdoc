== How to get set up:
    git clone git@github.com:rpearce/SCGOPrimaryTweets.git
    cd SCGOPrimaryTweets
    bundle install

== Run all major cities in charleston
    ruby run.rb

== Run individual city
    irb
    require './app.rb'
    blah = CandidateTweets.new('charleston')
    blah.get_tweets
